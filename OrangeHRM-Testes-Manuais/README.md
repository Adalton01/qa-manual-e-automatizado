# OrangeHRM - Testes Manuais

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar testes manuais utilizando o sistema OrangeHRM, aplicando técnicas de Quality Assurance (QA) em um ambiente real.

Durante os testes foram executados cenários de validação funcional, usabilidade, navegação e validação de entradas, simulando atividades realizadas por um Analista de QA Júnior.

---

## Ambiente de Teste

- Sistema: OrangeHRM Demo
- Navegador: Google Chrome
- Tipo de teste: Manual
- Plataforma: Web

---

## Escopo dos Testes

Foram realizados testes nos seguintes cenários:

- Login com credenciais válidas
- Login com credenciais inválidas
- Validação de campos obrigatórios
- Validação de campos contendo apenas espaços
- Recuperação de senha
- Navegação utilizando teclado (Tab)
- Testes de copiar e colar
- Validação de mensagens apresentadas ao usuário
- Testes com caracteres especiais
- Testes com entradas numéricas
- Testes com textos longos
